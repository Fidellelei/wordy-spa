# Wordly SPA Dictionary App

## Description
Wordly is a Single Page Application (SPA) dictionary web app built with HTML, CSS, and JavaScript. It allows users to search for words, view definitions, pronunciation, synonyms, examples, and save favorite words.

## Features

- Search any English word
- Live API dictionary data
- Display:
  - Word
  - Pronunciation
  - Definition
  - Example usage
  - Part of speech
  - Synonyms
  - Audio pronunciation
- Save favorites with localStorage
- Click saved favorite to search again
- Dark mode toggle
- Responsive layout
- Error handling

## Technologies Used

- HTML
- CSS
- JavaScript 
- Fetch API
- Local Storage
